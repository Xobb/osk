Deletes spam comments older than 1 month at speed 100 comments per hour.

== Description ==

Absolutely no configuration. The comments are deleted permanently, there's no way to restore them except the database backup.
Github repo: https://github.com/Xobb/osk

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
